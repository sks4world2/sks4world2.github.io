Cassandra NoSQL Database

Steps to install Cassandra NoSQL database and run CQL queries, Java programs in Cloudera.

These steps are specific to Cloudera VM and the steps may be slightly different for your setting, but the concepts remain the same.

    After logging into Cloudera Datastax VM...
  • Go to root folder in Ubuntu vm cd /root (you may need to use $su to get access to root
  • Review the tutorial here
  • Identify the tar bar dse-5.0.3.tar.gz (If not found, follow the tutorial to download and move to root folder)
  • Untar the tar file using $tar -xf dse-5.0.3.tar.gz
  • Change directory to /root/dse-5.0.3/bin
  • Start Cassandra NoSQL database by $./dse cassandra
  • Check if the database cluster is started by $./nodetool status
  • Check some images here
